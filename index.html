A bit about static analysis

Traditional static analysis has a few problems:

  • Multiple tools, all good at one thing
  • Variable integration points
  • No common report format
  • Speed (some are very, very slow)

This is where pipeline comes in

(with a bit of help from GeekOn)

OWASP pipeline is a tool for running multiple analysis tools and aggregating the results.


Developed in partnership with Groupon AppSec, pipeline was envisioned as a way to easily scan releases in an automated way.


Pipeline is ideal for integration into larger tools, which brings us to...

Introducing... Codeburner!

What is Codeburner?

Codeburner builds on OWASP pipeline. By providing a unified interface to view and act on the results of various tools, it answers the question: "I've got all these static analysis results... now what?"


Key Features:

  • Tight integration with Groupon services
  • Highly scalable, asynchronous scanning with multiple OSS and commercial tools
  • Advanced false positive filtering
  • Integrates with JIRA and GitHub for publishing issues
